## Sistema de Gestão de Alunos (CRUD)
Este projeto está sendo desenvolvido para o Hackathon, com o objetivo de entregar uma aplicação funcional para o gerenciamento de estudantes, integrando um sistema de autenticação e uma interface moderna.

⚠️ Status do Projeto: Em desenvolvimento.

## Sobre o Projeto
A aplicação foca em um ciclo completo de CRUD (Create, Read, Update, Delete) para o cadastro de alunos, protegida por um sistema de login para garantir a integridade dos dados.

## Tecnologias Utilizadas
`Front-end`
HTML5: Estruturação semântica e acessível.

CSS3: Estilização avançada com foco em:

Flexbox: Utilizado para o alinhamento dinâmico de componentes e elementos de navegação.

CSS Grid: Implementado para a estruturação robusta de layouts, dashboards e tabelas de dados.

`Back-end & Banco de Dados`
PHP: Lógica de negócio, controle de sessões e processamento de formulários.

MySQL: Banco de dados relacional para persistência de dados de alunos e usuários.

## Funcionalidades em Implementação
Sistema de Login: Autenticação segura para acesso ao painel.

CRUD de Alunos:

Cadastro de novos estudantes.

Listagem e consulta.

Edição de registros existentes.

Exclusão de registros.

Interface Responsiva: Design adaptável utilizando estratégias modernas de Grid e Flexbox.
